# Fun Games

A small collection of browser-based mini-games built with vanilla HTML5 Canvas and JavaScript. Open `index.html` in a modern browser — no build step required.

## Games

- **Highway Dash** — dodge oncoming traffic and grab coins while the speed ramps up
- **Whack-a-Mole** — tap the moles 🐹 as fast as you can and skip the bees 🐝
- **Add It Up!** — quick-fire addition & subtraction practice
- **Piano Memory** — Simon-style melody recall on a 7-key piano
- **Bus Route Rush** — drive a bus around a city grid, pick up colored passengers, and drop them at the matching stops. Chain deliveries to build a combo multiplier!

## Bus Route Rush controls

- **Arrow keys** or **WASD**: drive one block
- **Swipe** on the grid (mobile) or use the on-screen D-pad

## Layout

- `index.html` &ndash; game picker with per-game leaderboards
- `style.css` &ndash; shared picker styling
- `highway-dash/`, `whack-a-mole/`, `add-it-up/`, `piano/`, `bus-route-rush/` &ndash; each game's HTML, CSS, and JS
