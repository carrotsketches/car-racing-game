# Highway Dash

A simple browser-based car racing game built with vanilla HTML5 Canvas and JavaScript.

## Play

Open `index.html` in a modern browser. No build step required.

## Controls

- **Arrow keys** or **A / D**: Steer left/right
- **Arrow Up / W**: Accelerate
- **Arrow Down / S**: Slow down
- **Space**: Start / restart after a crash

## Goal

Dodge oncoming traffic for as long as possible. Your score climbs with every car you pass, the speed ramps up over time, and your best run is stored locally.

## Files

- `index.html` &ndash; page layout and HUD
- `style.css` &ndash; styling
- `game.js` &ndash; game loop, input handling, rendering
